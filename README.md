# MapViewer
Show geographical data on a map

`docker run -p 80:80 -d --name mapviewer`
## Running

## Base layers
- CBS (OpenStreetMap SE)
- MapBox
- MapQuest
- OpenStreetMap

## Overlays
- CBS buurten (2015)
- CBS buurten labels (2014)
- BAG panden
- BAG verblijfsobject
- BAG ligplaats
- BAG woonplaats
- 3D gebouwen

## Searching/Geocoding
- OpenStreetMap
- Funda (InternalId, TinyId)
