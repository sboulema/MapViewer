# MapViewer
Show geographical data on a map

## Running
`docker run -p 80:80 -d --name mapviewer`

## Base layers
- BRT
- MapBox
- MapQuest
- OpenStreetMap

## Overlays
- CBS buurten
- CBS buurten labels
- BAG panden
- BAG verblijfsobject
- BAG ligplaats
- BAG woonplaats
- 3D gebouwen

## Searching/Geocoding
- OpenStreetMap
- Funda (InternalId, TinyId)
